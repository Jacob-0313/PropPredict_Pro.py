🏠 PropPredict Pro
AI Real Estate Price & Investment Advisor
PropPredict Pro is an AI-powered real estate analytics application designed to predict property prices and evaluate investment potential. The system leverages machine learning models to provide accurate price estimates and an investment score based on property features.

The application provides an interactive dashboard that allows users to input property details and receive real-time predictions along with visual insights.

🎯 Objectives

Predict real estate property prices using Machine Learning

Evaluate investment opportunities with an AI-based scoring system

Provide interactive visualization of property data

Enable data-driven decision-making for investors
